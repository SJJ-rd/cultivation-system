# 生活轉化修持系統｜最終道場版

## 建議新 Repo 名稱
cultivation-system

## 上線網址範例
https://sjj-rd.github.io/cultivation-system/

## 功能
- 入壇畫面
- 道場儀式感 UI
- 叩首計數與目標進度
- 每日定課紀錄
- 儀軌分段導讀
- 靜坐三分鐘倒數
- 回向完成提示
- 占察善惡業報觀照
- 功德值與修行位階
- 修持歷史
- JSON 匯出 / 匯入
- PWA 可安裝

## GitHub Pages 部署
1. 建立新 repo：cultivation-system
2. 將本資料夾所有檔案上傳到 repo 根目錄
3. Settings → Pages
4. Source 選 Deploy from branch
5. Branch 選 main / root
6. 開啟 https://sjj-rd.github.io/cultivation-system/

## 備註
音效使用 Web Audio API 產生，不需額外 mp3 檔。
占察功能定位為修持反省工具，不作迷信斷言。
